Why SysMedic?

Every existing tool is a monitor (htop, btop, Mission Center), a cleaner (BleachBit, Stacer) or an expert CLI (systemd-analyze, smartctl). Nobody diagnoses, explains and fixes safely. SysMedic does all five steps of a doctor's visit — and never runs as root: privileged fixes go through a polkit-authorized helper that accepts a fix id only and rebuilds the plan, so nothing can be injected.

  • Weighted 0–100 health score
  • CPU, RAM, swap, disks, thermal, battery
  • SMART disk health & SSD wear
  • Services, processes, boot analysis
  • Security audit: firewall, SSH, exposed ports
  • Bilingual offline explanations (en/ar)
  • Safe fixes with preview & undo
  • Disk-usage treemap analyzer
  • Scheduled checkups & desktop alerts
  • HTML / Markdown / JSON / PDF reports
  • Optional AI deep-explain (bring your own key)
  • GTK4/libadwaita — automatic dark/light

Try the CLI

sudo apt install ./sysmedic_*.deb

sysmedic checkup                       # colored 0–100 report
sysmedic explain storage.disk_nearly_full --lang ar
sysmedic explain storage.disk_nearly_full --deep   # optional: AI, opt-in
sysmedic fix fix.apt_clean --dry-run   # preview a safe fix
sysmedic disk ~                        # largest folders
sysmedic schedule daily                # follow-up checkups
